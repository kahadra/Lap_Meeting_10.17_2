Parameter α and Trade-off

데이터 충실도와 정규화 사이의 줄다리기

파라미터 α는 에너지 함수 E(R) = Ed + αEs 에서 두 항의 균형을 조절하는 매우 중요한 역할을 합니다.

α 값 효과 (어떤 항을 더 중요하게 생각하는가?) 결과 영상
α → 0 (작음) 데이터 항 (Ed)을 우선시합니다. (R ≈ I)
→ 정규화의 영향이 거의 없습니다. 		노이즈 잔존
α 중간 데이터 항과 정규화 항의 균형을 맞춥니다.
→ 적절한 노이즈 제거와 디테일 보존. 		균형 잡힌 결과
α → ∞ (큼) 정규화 항 (Es)을 우선시합니다. (∇R ≈ 0)
→ 원본 데이터의 내용은 무시하고 무조건 부드럽게 만듭니다. 		과도한 블러
