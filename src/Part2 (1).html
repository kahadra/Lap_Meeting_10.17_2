Taylor Approximation

복잡한 비선형 함수를 간단한 선형 함수로 근사하기

기본 원리

어떤 점 a 근처에서, 복잡한 함수 f(x)는 그 점에서의 함수값과 미분값들을 이용해 다항식으로 근사할 수 있습니다.

f(x) ≈ f(a) + f'(a)(x-a) + 1⁄2f''(a)(x-a)2

변분법에서는 주로 1차항까지만 사용하여 비선형 문제를 국소적인 선형 문제로 바꾸어 풉니다.

적용 예시: Optical Flow

움직이는 영상에서 '밝기 항등 가정'을 선형화하는 과정입니다.

I(x+u, y+v, t+1) = I(x,y,t)

Taylor 1차 근사

I(x,y,t) + Ixu + Iyv + It ≈ I(x,y,t)

정리하면

Ixu + Iyv + It ≈ 0
