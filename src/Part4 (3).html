The Linear System

모든 픽셀에 대한 방정식을 행렬 형태로 표현하기

이산화된 방정식을 행렬과 벡터로 표현

이미지의 모든 픽셀에 대해 이산화된 방정식을 적용하고, 픽셀 값들을 하나의 긴 벡터로 생각하면,
전체 문제는 다음과 같은 거대한 선형 연립방정식으로 표현됩니다.

(I + αL)r = i

r : 우리가 찾고자 하는 복원 영상의 모든 픽셀 값을 담은 벡터 (Unknown)

i : 관측된 영상의 모든 픽셀 값을 담은 벡터 (Known)

L : 모든 픽셀에 대해 2D Laplacian 연산을 수행하는 거대한 행렬

I : 항등 행렬 (Identity Matrix)
