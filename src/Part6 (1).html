Summary (1/2): From Problem to PDE

문제 정의부터 미분방정식 유도까지의 과정

문제 정의: 역문제 (Inverse Problem)

관측 영상 I = R + n 으로부터 원본 영상 R을 추정하는 문제로, 직접 풀 수 없어 최적화 기법이 필요합니다.

에너지 정식화: 데이터 항 + 정규화 항

에너지 E = Ed + αEs 를 정의하고, 이를 최소화하는 해를 찾습니다. (데이터 충실도 vs. 사전 지식)

Tikhonov (L2) 정규화: 평활화 (Smoothing)

정규화 항으로 ∫|∇R|2 를 사용하여 부드러운 해를 유도합니다.

해법: 오일러-라그랑주 방정식

에너지 최소화 조건으로부터 편미분방정식 (R - I) - α∇2R = 0 을 유도합니다.
